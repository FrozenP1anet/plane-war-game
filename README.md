# 飞机大战

一个基于原生 `HTML + CSS + JavaScript + Canvas` 开发的纵版飞机大战小游戏。项目保留了经典街机射击的即时反馈，同时加入了多武器切换、特殊敌机、阶段事件、普通 Boss、第三关终局 Boss 和结算面板等扩展玩法。

## 项目特点

- 支持 `闯关模式` 与 `无尽模式`
- 支持设置页调整全局难度倍率
- 内置帮助页，使用项目内 `assets` 插图说明玩法
- 包含多种武器系统：默认单发、速射、扩散、重炮、集束导弹
- 包含特殊敌人机制：突袭机、炸弹机、补给机、陨石带
- 包含普通 Boss 多模式攻击与第三关终局 Boss 二阶段战斗
- 提供成绩结算与阶段信息展示

## 运行方式

这是一个纯前端项目，不依赖构建工具。直接用浏览器打开 [`index.html`](./index.html) 即可开始游戏。

更推荐的本地运行方式是使用一个静态文件服务器，例如：

```powershell
npx serve .
```

然后在浏览器中打开命令行输出的本地地址。

## 操作说明

### 键盘操作

- `WASD` / 方向键：移动战机
- `H`：按住使用速射
- `J`：按住使用扩散
- `K`：按住蓄力，松开发射重炮
- `M`：启动集束导弹
- `Space`：暂停 / 继续

### 触屏操作

页面底部提供方向按钮和武器按钮，适合触屏设备试玩。

## 游戏模式

### 闯关模式

- 从第一关开始推进
- 当前版本共 3 关
- 前两关以敌机波次和特殊事件为主
- 第三关后期进入终局 Boss 战
- 击败终局 Boss 后，需要主动接触黑洞完成撤离结算

### 无尽模式

- 没有时间上限
- 难度会随生存时间和分数持续增长
- 每隔一段时间进入新的阶段事件
- 当前事件包括突袭风暴、轰炸走廊、陨石暴雨、补给争夺、火力试炼

## 武器系统

- `单发`：默认武器，无限弹药，适合稳定清理普通敌人
- `速射`：高频输出，适合压制高速目标和 Boss
- `扩散`：覆盖面更大，适合清杂和处理中距离压境敌群
- `重炮`：可蓄力，适合对高血量目标进行爆发输出
- `导弹`：短时间自动锁定敌人，适合混战时提升容错

武器补给和生命补给会在战斗过程中掉落，合理保留关键弹药通常比立刻用掉更重要。

## 敌人与 Boss

### 常见敌人

- `普通敌机`：基础单位，数量多时会压缩走位空间
- `突袭机`：高速切入并留下危险尾迹
- `炸弹机`：被击毁后会产生范围爆炸
- `补给机`：会为附近敌人提供回复
- `陨石`：高耐久障碍物，迫使玩家调整站位

### 普通 Boss

普通 Boss 会在流程中阶段性登场，并切换不同攻击模式，例如：

- `狙击模式`
- `扇扫模式`
- `连射模式`

### 终局 Boss

第三关后期会出现终局 Boss，拥有：

- 更高生命值和更复杂的攻击组合
- 速射、扩散、重炮、冲刺、召唤等招式
- 二阶段强化机制
- 被击败后生成黑洞完成关卡结算

## 设置与存档

- 设置页可通过滑块调整难度倍率
- 当前设置会保存在浏览器本地存储中
- 存储键为 `plane-war-settings-v1`

## 项目结构

```text
.
├─ assets/       # 游戏插图与界面资源
├─ config.js     # 数值、关卡、敌人、武器与资源配置
├─ game.js       # 游戏主逻辑、状态管理、渲染与交互
├─ index.html    # 页面结构、菜单、帮助页与画布容器
├─ styles.css    # 页面样式与界面布局
└─ dist/         # 本地保留的打包产物，不纳入 Git 跟踪
```

## 开发说明

- 游戏核心基于 Canvas 渲染
- 主要游戏状态与行为逻辑集中在 [`game.js`](./game.js)
- 全局参数集中在 [`config.js`](./config.js)，适合继续调数值和平衡性
- 帮助页与主界面结构集中在 [`index.html`](./index.html)

如果你准备继续扩展这个项目，比较适合的方向包括：

- 新增关卡或阶段事件
- 为普通 Boss 增加更多弹幕组合
- 增加新武器或被动效果
- 优化移动端触控体验
- 增加音效、背景音乐和更丰富的视觉特效
